# Harper for Notepad++

A color scheme inspired by the art of [Charley Harper](http://en.wikipedia.org/wiki/Charley_Harper) (1922-2007)

![Screenshot](https://raw.github.com/idleberg/Harper-Notepad-plus-plus/master/images/screenshot.png)

## Installation

1. Download an `.xml` file of choise
2. Copy the file to `%APPDATA%/Notepad++/themes`
3. Select the theme from the preferences menu
